Opis szczegółowy

Klasa szkieletu aplikacji.

Zadaniem tej klasy jest inicjalizacja oraz niszczenie całego frameworka oraz zarządzanie główną pętlą programu.
Należy utworzyć obiekt tej klasy w głównej metodzie programu WinMain oraz wywołać jej jedyną publiczną metodę CFramework::Run.

GLboolean CFramework::InitalizationGL (  )  [private]

Główna metoda inicjalizująca wstępne parametry OpenGL-a.

Wewnątrz tej metody wywoływana jest także metoda CFramework::CreateGameControl.

Zwraca:
GL_TRUE w przypadku powodzenia inicjalizacji. GL_FALSE w przypadku wystąpienia jakiegoś błędu.

GLboolean CFramework::CreateGameControl (  )  [private]

Metoda tworząca oraz inicjalizująca obiekt klasy CGameControl.

Zwraca:
GL_TRUE w przypadku powodzenia inicjalizacji. GL_FALSE w przypadku wystąpienia jakiegoś błędu.

GLboolean CFramework::DrawGLScene (  )  [private]

Główna metoda, od której rozpoczyna się rysowanie całej sceny.

Wewnątrz tej metody wywoływana jest nie mniej ważna metoda CGameControl::Draw.

Zwraca:
GL_TRUE w przypadku powodzenia. GL_FALSE w przypadku wystąpienia jakiegoś błędu.

GLvoid CFramework::Destroy ( GLboolean  bKillGLWindow  )  [private]

Metoda usuwająca całe okno frameworka oraz obiekt klasy CGameControl.

Parametry:
[in] bKillGLWindow Podając jako GL_FALSE, zniszczony zostanie tylko obiekt klasy CGameControl. Podając GL_TRUE, oprócz zniszczenia CGameControl, zniszczone zostanie także okno apliakcji.
