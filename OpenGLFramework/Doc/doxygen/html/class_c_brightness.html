Opis szczegółowy

Klasa kontroli jasności ekranu.

Zadaniem tej klasy jest rysowanie prostokąta na całym ekranie z odpowienio ustawioną przezroczystością oraz kolorem dla uzyskania efektu rozjaśnienia lub przyciemnienie ekranu. Obiekt tej klasy jest już tworzony w CGameControl oraz „rysowanie jasności” także jest już wywoływane w CGameControl::Draw.

GLvoid CBrightness::SetBrightness ( GLint  iBrightness = 0  ) 

Metoda ustawiająca wartość jasności ekranu.

Parametry:
[in] iBrightness Wartość jasności od -255 do 255, gdzie -255 to całkowite przyciemnienie (obraz będzie czarny) a 255 to całkowite rozjaśnienie (obraz będzie biały). Podając 0 (wartość domyślna) obraz będzie rysowany w oryginalnej jasności.

Dokumentacja atrybutów składowych

GLint CBrightness::m_iBrightness [private]

Zmienna przechowująca aktualną wartość jasności.

Wartość jasności od -255 do 255, gdzie -255 to całkowite przyciemnienie (obraz będzie czarny) a 255 to całkowite rozjaśnienie (obraz będzie biały). W zmiennej ten ustotna jest możliwość usawienia wartości ujemnej, która wskazuje na przyciemnienie obrazu.

GLubyte CBrightness::m_ubBrightness [private]

Zmienna przechowująca aktualną wartość przezroczystości dla rysowania.

Wartość przeroczystości od 0 do 255, gdzie 0 to całkowita przezroczystość (jasność bez z mian) a 255 to całkowite rozjaśnienie lub przyciemnienie.
