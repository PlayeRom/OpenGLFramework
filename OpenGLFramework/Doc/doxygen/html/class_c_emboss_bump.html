Opis szczegółowy

Klasa tworzenia efektu tłoczonego mapowania wybojów.

Zadaniem tej klasy jest obliczenie wartości tangent i binormal dla przesunięcia koordynatów tekstur, utworzenie tablicy nowych, przesuniętych koordynatów tekstur, wsparcie dla rysowania mapowania wybojów.
