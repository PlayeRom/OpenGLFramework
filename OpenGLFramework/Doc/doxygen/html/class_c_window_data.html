Opis szczegółowy

Klasa okna aplikacji.

Klata ta odpowiada za tworzenie i usuwanie okna oraz co ważne przechowuje różne informacje takie jak:

  • rozdzielczość ekranu,
  • głębia kolorów,
  • kolor tła,
  • nazwę okna i klasy okna,
  • aktywność antyaliasingu, trybu pełnoekranowego, wyświetlanie kursora

GLboolean CWindowData::SaveSettingFile ( GLboolean  bDefaultSave = GL_FALSE  ) 

Metoda zapisująca plik konfiguracyjny.

Do pliku konfiguracyjnego zostanie zapisana struktura CWindowData::m_sSettingFile.

Parametry:
[in] bDefaultSave Jeżeli podamy GL_TRUE to zostaną zapisane ustawienia domyślne a nie rzeczywiste jakie są aktualnie przechowywane w CWindowData::m_sSettingFile. Ustawienia domyślne zapisywane są tylko przy pierwszym uruchomieniu aplikacji. Dalej należy zapisywać ustawienia rzeczywiste czyli skorzystać w domyślnej wartości parametru: GL_FALSE.
Zwraca:
GL_TRUE w przypadku powodzenia zapisu pliku konfiguracyjnego. GL_FALSE jeżeli wystąpił jakiś błąd.
Zobacz również:
LoadSettingFile, SETTING_FILE_NAME, m_sSettingFile.

GLvoid CWindowData::SetDeviceOnFullscreen (  )  [private]

Metoda ustawiająca tryb urządzenia graficznego na tryb pełnoekranowy.

Próba ustawienia trybu pełnoekranowego będzie realizowana tylko wtedy jeżeli flaga SSettingFile::bFullScreen ustawiona będzie na GL_TRUE.
Jeżeli próba ustawienia urządzenia graficznego na tryb pełnoekranowy zakończy się niepowodzeniem, aplikacja zostanie uruchomiona w trybie okna oraz flaga SSettingFile::bFullScreen zostanie ustawiona na GL_FALSE.
