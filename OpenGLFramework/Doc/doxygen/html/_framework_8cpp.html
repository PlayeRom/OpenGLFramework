Funkcja przetwarzająca komunikaty systemowe przekazywane do okna aplikacji.

Funkcja ta przetwarza niezbędne komunikaty przekazywane przez system i przekazuje je dalej do klasy CGameControl gdzie zostaną odpowiednio obsłużone reakcje na dane komunikaty.

Parametry:
[in] hWnd Uchyt okna aplikacji
[in] uMsg Identyfikator komunikatu jaki będzie przetwarzany.
[in] wParam Parametr komunikatu.
[in] lParam Parametr komunikatu.
Zwraca:
0 jeżeli komunikat został przetworzony.
