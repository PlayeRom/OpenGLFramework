#define LNG_LINE ( x   )     CMultiLanguage::GetInstance()->GetLangText( x )

Makro pobrania wskazanej linii tekstu z pliku językowego.

Makro to wywołuje metodę CMultiLanguage::GetLangText, zwracającą łańcuch znaków z pliku językowego.

Parametry:
[in] x Numer linii z pliku tekstowego, który chcemy pobrać, np. w celu wyświetlenia. Numery linii liczone są od 0.
Zwraca:
Łańcuch znaków zapisany w podanej linii pliku *.lng.
