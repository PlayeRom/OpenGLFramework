CWindowData::SSettingFile

Dokumentacja struktury CWindowData::SSettingFile

#include <WindowData.h>

Lista wszystkich składowych.

Opis szczegółowy

Struktura przechowująca ustawienia aplikacji.

Są tu przechowywane różne opcje, które mogą być modyfikowane w trakcie działania apliakcji. Jeżeli planujemy dodać więcej opcji konfiguracyjnych (np. włączenie/wyłączenie dźwięku, wybór języka itp.), to należy dodać je do struktury SSettingFile. Zapewni to automatyczny zapis i odczyt tychże ustawień.
Dodając z czasem nowe pola należy także pamiętać o zmianie numerowania wersji pliku. Dzięki temu unikniemy sytuacji wczytania przez użytkownika starszego pliku konfiguracyjnego, który będzie zawierał starszą wersję struktury. Dla zmiany wersji pliku konfiguracyjnego, patrz plik define.h.
