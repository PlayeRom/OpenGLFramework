Dokumentacja składowych wyliczanych

enum CConsole::EConsoleMsgType

Typ wyliczeniowy reprezentujący rodzaje wiadomości.

Każdą informację wyświetlaną w konsoli można określić danym typem przypisując jej odpowiedni kolor.

Wartości wyliczeń:
ENormal  Tekst normalny w kolorze białym.
EWarning  Ostrzeżenie w kolorze żółtym.
EError  Błąd w kolorze czerwonym.
ESuccess  Sukces w kolorze zielonym.
