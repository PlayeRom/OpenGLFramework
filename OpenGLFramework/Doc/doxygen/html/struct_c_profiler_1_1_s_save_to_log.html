CProfiler::SSaveToLog

Dokumentacja struktury CProfiler::SSaveToLog

Lista wszystkich składowych.

Opis szczegółowy

Struktura przechowująca wszelkie informacje jakie zostaną zapisaane do pliku.

Struktura przechowuje takie informacje jak komentarz wywołania metody CProfiler::Begin, czas jaki upłynął między wywołaniami CProfiler::Begin a CProfiler::End oraz ilość wcięć dla lepszej czytelności loga.

 Czy jest w użyciu?
GLint iIndentNumber
 Ilość wcięć.
TCHAR lpComment [MAX_SIZE_OF_STRING]
 Komentarz jaki został podany przy wywołaniu metody CProfiler::Begin.
unsigned long ulTimeMs
 Czas w milisekundach jaki upłynął między wywołaniem CProfiler::Begin a CProfiler::End.
